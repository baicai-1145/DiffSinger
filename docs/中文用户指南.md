## 项目简介

DiffSinger 是一个基于浅层扩散机制的歌声合成系统，由 OpenVPI 团队维护和改进。它是对原始论文《DiffSinger: Singing Voice Synthesis via Shallow Diffusion Mechanism》的重构和增强版本，具有以下特点：

### 核心优势

- **更清晰的代码结构**：移除了冗余文件，重新组织了代码架构
- **更好的音质**：采样率从 24 kHz 提升到 44.1 kHz
- **更高的保真度**：集成了改进的声学模型和扩散采样加速算法
- **更强的可控性**：引入了音高、能量、气息等参数的预测和控制
- **生产兼容性**：功能设计符合生产部署和歌声合成社区的需求

### 系统架构

DiffSinger 采用两阶段的训练和推理架构：

1. **变化模型 (Variance Model)**：预测音高、能量、气息等声学参数
2. **声学模型 (Acoustic Model)**：基于音素序列和声学参数生成梅尔频谱图

## 基础概念

### 语言标签

每种处理的语言都应该在配置文件中有唯一的标签。**强烈建议使用 ISO 639 语言代码作为语言标签**：

- `zh` 或 `zho`：中文（`cmn` 特指普通话）
- `ja` 或 `jpn`：日语
- `en` 或 `eng`：英语
- `yue`：粤语

### 音素系统

音素是字典和标签的基础部分，分为两种类型：

#### 语言特定音素
如果有多种语言，所有语言特定的音素都会以语言名称为前缀：
- `zh/a`、`ja/o`、`en/eh`（完整名称）
- `a`、`o`、`eh`（简短名称，仅在特定语言上下文中有意义）

#### 全局音素
不属于任何语言的音素：
- `SP`：空格
- `AP`：送气音
- 其他用户定义标签：`EP`、`GS`、`VF` 等

DiffSinger 期望的数据集结构如下：

## 数据准备

### 音频文件要求

- **格式**：WAV 文件
- **采样率**：44100 Hz（推荐）或更高
- **位深度**：16-bit 或 24-bit
- **声道**：单声道（推荐）或立体声
- **质量**：清晰的人声，最小化背景噪音
- **长度**：建议每个文件 10-60 秒

### 转录文件格式

转录文件应为 CSV 格式，包含以下列：

```csv
name,ph_seq,ph_dur,ph_num,note_seq,note_dur,note_slur,f0_seq,f0_timestep,spk_id
song1,zh/ni zh/h zh/ao SP zh/m zh/a,0.15 0.10 0.20 0.05 0.15 0.25,6,C4 C4 D4 rest E4 E4,0.25 0.30 0.15 0.05 0.40,0 0 1 0 1,220.5 223.1 293.7 0.0 329.6,0.01,0
```

#### 列说明：

- `name`：音频文件名（不含扩展名）
- `ph_seq`：音素序列（空格分隔）
- `ph_dur`：音素持续时间（秒）
- `ph_num`：音素数量
- `note_seq`：音符序列
- `note_dur`：音符持续时间
- `note_slur`：连音标记（0=否，1=是）
- `f0_seq`：基频序列（Hz）
- `f0_timestep`：基频时间步长
- `spk_id`：说话人 ID

### 数据预处理工具

#### 1. 自动音频分割

对于长音频文件，可以使用以下工具进行分割：

```python
import librosa
import soundfile as sf

def split_audio(input_file, output_dir, segment_length=30):
    """将长音频分割为固定长度的片段"""
    y, sr = librosa.load(input_file, sr=44100)
    segment_samples = segment_length * sr
    
    for i in range(0, len(y), segment_samples):
        segment = y[i:i+segment_samples]
        if len(segment) > sr * 5:  # 至少5秒
            output_file = f"{output_dir}/segment_{i//segment_samples:03d}.wav"
            sf.write(output_file, segment, sr)
```

#### 2. 音频质量检查

```python
import librosa
import numpy as np

def check_audio_quality(file_path):
    """检查音频质量"""
    y, sr = librosa.load(file_path, sr=44100)
    
    # 检查音量
    rms = np.sqrt(np.mean(y**2))
    db = 20 * np.log10(rms + 1e-6)
    
    # 检查静音
    silence_threshold = 0.01
    silence_ratio = np.sum(np.abs(y) < silence_threshold) / len(y)
    
    # 检查剪切失真
    clipping_ratio = np.sum(np.abs(y) > 0.99) / len(y)
    
    print(f"文件：{file_path}")
    print(f"RMS 音量：{db:.2f} dB")
    print(f"静音比例：{silence_ratio:.2%}")
    print(f"剪切失真比例：{clipping_ratio:.2%}")
    
    return db > -40 and silence_ratio < 0.3 and clipping_ratio < 0.01
```

## 数据预处理

数据预处理（二进制化）是训练前的必要步骤，它将原始数据转换为训练所需的格式。

### 预处理命令

```bash
python scripts/binarize.py --config config_variance.yaml
```

### 预处理过程详解

#### 1. 音频特征提取

- **梅尔频谱图**：提取 80 维梅尔频谱图
- **基频 (F0)**：使用 RMVPE 提取基频轨迹
- **能量**：计算每帧的能量值
- **谱质心**：计算频谱质心特征

#### 2. 文本处理

- **音素对齐**：将音素与音频帧对齐
- **音素编码**：将音素转换为数字索引
- **语言标识**：为多语言模型添加语言 ID

#### 3. 数据验证

预处理过程中会进行多项验证：

```python
# 检查音频长度与标注一致性
def validate_duration(audio_length, annotation_length, tolerance=0.1):
    """验证音频长度与标注的一致性"""
    return abs(audio_length - annotation_length) < tolerance

# 检查音素序列完整性
def validate_phonemes(ph_seq, dictionary):
    """验证音素序列的完整性"""
    for ph in ph_seq:
        if ph not in dictionary and ph not in ['SP', 'AP']:
            return False, f"未知音素：{ph}"
    return True, "音素序列有效"
```

1. **变化模型训练**：学习预测音高、能量等声学参数
2. **声学模型训练**：基于声学参数生成梅尔频谱图

### 变化模型训练

#### 1. 启动训练

```bash
python scripts/train.py --config config_variance.yaml --exp_name variance_singer1 --reset
```

参数说明：
- `--config`：配置文件路径
- `--exp_name`：实验名称（影响保存路径）


- **持续时间损失 (Duration Loss)**：时长预测准确性
- **能量损失 (Energy Loss)**：能量预测准确性

## 模型推理

### 推理流程

模型推理基于 DS 文件格式，包含音符信息和歌词。

### DS 文件格式

DS 文件是 JSON 格式，包含以下结构：

```json
{
  "segments": [
    {
      "offset": 0.0,
      "ph_seq": "zh/wo zh/ai zh/ni",
      "ph_dur": [0.5, 0.3, 0.7],
      "note_seq": ["C4", "D4", "E4"],
      "note_dur": [0.5, 0.3, 0.7],
      "note_slur": [false, false, false]
    }
  ],
  "spk_id": 0,
  "gender": 0,
  "velocity": 100
}
```

### 变化模型推理

#### 1. 基本推理命令

```bash
python scripts/infer.py variance input.ds --exp variance_singer1 --out output_var.ds
```

#### 2. 推理参数

```bash
python scripts/infer.py variance input.ds \
  --exp variance_singer1 \
  --out output_var.ds \
  --spk_id 0 \
  --gender 0 \
  --velocity 100
```

参数说明：
- `--spk_id`：说话人 ID
- `--gender`：性别调整（-1.0 到 1.0）
- `--velocity`：力度调整（0 到 127）

#### 3. 高级推理选项

```bash
python scripts/infer.py variance input.ds \
  --exp variance_singer1 \
  --out output_var.ds \
  --pitch_shift 2 \        # 音高偏移（半音）
  --energy_scale 1.2 \     # 能量缩放
  --breathiness 0.5        # 气息程度
```

### 声学模型推理

#### 1. 基本推理命令

```bash
python scripts/infer.py acoustic input_var.ds --exp acoustic_singer1 --out output.wav
```

#### 2. 扩散采样参数

```bash
python scripts/infer.py acoustic input_var.ds \
  --exp acoustic_singer1 \
  --out output.wav \
  --steps 50 \              # 扩散步数
  --method pndm \           # 采样方法
  --eta 0.0                 # 随机性控制
```

采样方法选项：
- `ddim`：DDIM 采样（快速但质量一般）
- `pndm`：PNDM 采样（平衡速度和质量）
- `dpm-solver++`：DPM-Solver++（高质量）
- `unipc`：UniPC（最新算法）

#### 3. 声码器选择

```bash
python scripts/infer.py acoustic input_var.ds \
  --exp acoustic_singer1 \
  --out output.wav \
  --vocoder nsf-hifigan \   # 声码器类型
  --vocoder_ckpt path/to/vocoder.ckpt
```

声码器选项：
- `nsf-hifigan`：高质量神经声码器
- `pc-ddsp`：基于 DDSP 的声码器

### 完整推理流程

#### 1. 创建 DS 文件

```python
import json

def create_ds_file(lyrics, notes, durations, output_path):
    """创建 DS 文件"""
    ds_data = {
        "segments": [{
            "offset": 0.0,
            "ph_seq": " ".join(lyrics),
            "ph_dur": durations,
            "note_seq": notes,
            "note_dur": durations,
            "note_slur": [False] * len(notes)
        }],
        "spk_id": 0,
        "gender": 0,
        "velocity": 100
    }
    
    with open(output_path, 'w', encoding='utf-8') as f:
        json.dump(ds_data, f, ensure_ascii=False, indent=2)

# 示例使用
lyrics = ["zh/wo", "zh/ai", "zh/ni"]
notes = ["C4", "D4", "E4"]
durations = [0.5, 0.3, 0.7]
create_ds_file(lyrics, notes, durations, "song.ds")
```

#### 2. 完整推理脚本

```bash
#!/bin/bash

# 设置变量
INPUT_DS="input_song.ds"
VAR_OUTPUT="variance_output.ds"
FINAL_OUTPUT="final_song.wav"
VAR_EXP="variance_singer1"
ACOUSTIC_EXP="acoustic_singer1"

# 步骤1：变化模型推理
echo "运行变化模型推理..."
python scripts/infer.py variance $INPUT_DS \
  --exp $VAR_EXP \
  --out $VAR_OUTPUT \
  --spk_id 0

# 步骤2：声学模型推理
echo "运行声学模型推理..."
python scripts/infer.py acoustic $VAR_OUTPUT \
  --exp $ACOUSTIC_EXP \
  --out $FINAL_OUTPUT \
  --steps 50 \
  --method pndm

echo "推理完成！输出文件：$FINAL_OUTPUT"
```

### 推理优化技巧

#### 1. 批量推理

```python
import os
import subprocess

def batch_inference(ds_files, var_exp, acoustic_exp, output_dir):
    """批量推理多个 DS 文件"""
    for ds_file in ds_files:
```

```python
def create_inference_session(model_path, device="cuda"):
```

### 部署示例

#### Python 推理接口

```python
import onnxruntime as ort
import numpy as np
import json

class DiffSingerONNXInference:
    def __init__(self, variance_path, acoustic_path, vocoder_path):
        """初始化 ONNX 推理引擎"""
        self.variance_session = ort.InferenceSession(variance_path)
        self.acoustic_session = ort.InferenceSession(acoustic_path)
        self.vocoder_session = ort.InferenceSession(vocoder_path)
        
    def predict_variance(self, ph_seq, ph_dur, note_seq, note_dur):
        """变化模型推理"""
        inputs = {
            "ph_seq": ph_seq.astype(np.int64),
            "ph_dur": ph_dur.astype(np.float32),
            "note_seq": note_seq.astype(np.int64),
            "note_dur": note_dur.astype(np.float32)
        }
        
        outputs = self.variance_session.run(None, inputs)
        return outputs[0], outputs[1], outputs[2]  # f0, energy, breathiness
        
    def predict_acoustic(self, ph_seq, f0, energy, breathiness):
        """声学模型推理"""
        inputs = {
            "ph_seq": ph_seq.astype(np.int64),
            "f0": f0.astype(np.float32),
            "energy": energy.astype(np.float32),
            "breathiness": breathiness.astype(np.float32)
        }
        
        outputs = self.acoustic_session.run(None, inputs)
        return outputs[0]  # mel_spectrogram
        
    def vocode(self, mel):
        """声码器推理"""
        inputs = {"mel": mel.astype(np.float32)}
        outputs = self.vocoder_session.run(None, inputs)
        return outputs[0]  # waveform
        
    def synthesize(self, ds_file_path, output_wav_path):
        """完整合成流程"""
        with open(ds_file_path, 'r', encoding='utf-8') as f:
            ds_data = json.load(f)
        
        segment = ds_data["segments"][0]
        
        # 准备输入数据
        ph_seq = np.array([self.ph_to_id(ph) for ph in segment["ph_seq"].split()])
        ph_dur = np.array(segment["ph_dur"])
        note_seq = np.array([self.note_to_id(note) for note in segment["note_seq"]])
        note_dur = np.array(segment["note_dur"])
        
        # 变化模型推理
        f0, energy, breathiness = self.predict_variance(ph_seq, ph_dur, note_seq, note_dur)
        
        # 声学模型推理
        mel = self.predict_acoustic(ph_seq, f0, energy, breathiness)
        
        # 声码器推理
        waveform = self.vocode(mel)
        
        # 保存音频
        import soundfile as sf
        sf.write(output_wav_path, waveform, 44100)
```

   ```bash
   pip install -r requirements.txt  # 再安装其他依赖
   ```

### 数据预处理问题

#### Q3: 音频时长不匹配

**问题**：`Duration mismatch: audio=10.5s, annotation=9.8s`

**解决方案**：
1. 检查音频文件是否损坏
2. 验证转录文件的时间标注
3. 调整容差设置：
   ```yaml
   binarization_args:
     duration_tolerance: 0.2  # 增加容差
   ```

#### Q5: 显存不足

**问题**：`RuntimeError: CUDA out of memory`

**解决方案**：
1. 减少批处理大小：
   ```yaml
   train_args:
     batch_size: 8  # 从 16 减少到 8
   ```

   ```yaml
   trainer_args:
   ```

2. 增加梯度裁剪：
   ```yaml
   clip_grad_norm: 0.5
   ```
3. 检查数据质量和标注准确性

```python
    """模型参数插值"""
    model1 = torch.load(model1_path)
    model2 = torch.load(model2_path)
    
    interpolated_state = {}
```

```python
def speaker_adaptation(base_model, speaker_data, learning_rate=1e-5):
```

### 音质增强技术

#### 1. 后处理滤波

```python
import scipy.signal as signal

def post_process_audio(waveform, sample_rate=44100):
    """音频后处理"""
    # 高通滤波去除低频噪音
    high_pass = signal.butter(4, 80, btype='high', fs=sample_rate, output='sos')
    waveform = signal.sosfilt(high_pass, waveform)
    
    # 轻微压缩提升动态
    waveform = np.tanh(waveform * 1.2) / 1.2
    
    # 去除直流偏移
    waveform = waveform - np.mean(waveform)
    
    return waveform
```

```python
def multiband_processing(waveform, sample_rate=44100):
    """多频带处理"""
    # 分离频带
    low_freq = signal.butter(4, 1000, btype='low', fs=sample_rate, output='sos')
    mid_freq = signal.butter(4, [1000, 8000], btype='band', fs=sample_rate, output='sos')
    high_freq = signal.butter(4, 8000, btype='high', fs=sample_rate, output='sos')
    
    low_band = signal.sosfilt(low_freq, waveform)
    mid_band = signal.sosfilt(mid_freq, waveform)
    high_band = signal.sosfilt(high_freq, waveform)
    
    # 分别处理
    low_band *= 0.9   # 略微衰减低频
    mid_band *= 1.1   # 增强中频
    high_band *= 1.05 # 轻微增强高频
    
    # 重新合成
    return low_band + mid_band + high_band
```

- **最低要求**: Python 3.8
- **推荐版本**: Python 3.9-3.10
